# 02 Patrón Concurrente

En un patrón concurrente, múltiples agentes analizan la misma entrada simultáneamente, cada uno aportando su propia experiencia. Una vez que todos los agentes completan su tarea, sus salidas se combinan en un resultado unificado. Esto es ideal para tareas que se benefician de múltiples puntos de vista trabajando al mismo tiempo, como análisis desde múltiples perspectivas, evaluación en conjunto o toma de decisiones colaborativa.
