# 04 Patrón de Chat Grupal

En un patrón de chat grupal, múltiples agentes participan en una conversación compartida, turnándose para aportar su experiencia. Un orquestador gestiona el flujo de la discusión, decidiendo qué agente habla a continuación y cuándo debe terminar la conversación. Esto es muy adecuado para planificación multifuncional, lluvia de ideas creativa o cualquier tarea donde diversas perspectivas necesitan interactuar y construir sobre las ideas de los demás de manera iterativa.
