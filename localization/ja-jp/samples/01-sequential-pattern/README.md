# 01 シーケンシャルパターン

シーケンシャルパターンでは、エージェントが定義されたパイプラインの中で順番に動作し、各エージェントの出力が次のエージェントへの入力となります。このアプローチは、コンテンツ作成ワークフロー、段階的なデータ変換、ステップバイステップの分析など、自然な進行に従うタスクに適しています。
