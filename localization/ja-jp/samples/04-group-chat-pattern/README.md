# 04 グループチャットパターン

グループチャットパターンでは、複数のエージェントが共有の会話に参加し、交代でそれぞれの専門知識を提供します。オーケストレーターが議論の流れを管理し、次にどのエージェントが発言するか、いつ会話を終了するかを決定します。これは、部門横断的な計画立案、クリエイティブなブレインストーミング、または多様な視点が相互作用し、反復的にアイデアを発展させる必要があるタスクに適しています。
