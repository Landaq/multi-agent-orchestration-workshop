# 02 동시 실행 패턴

동시 실행 패턴에서는 여러 에이전트가 동일한 입력을 동시에 분석하며, 각 에이전트가 자신만의 전문 지식을 활용합니다. 모든 에이전트의 작업이 완료되면 출력이 하나의 통합된 결과로 결합됩니다. 이 패턴은 다중 관점 분석, 앙상블 평가 또는 협업 의사 결정과 같이 여러 관점에서 동시에 작업하는 것이 유리한 작업에 이상적입니다.
