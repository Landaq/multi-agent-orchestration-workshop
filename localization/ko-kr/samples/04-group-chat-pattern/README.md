# 04 그룹 채팅 패턴

그룹 채팅 패턴에서는 여러 에이전트가 공유 대화에 참여하여 순서대로 자신의 전문 지식을 제공합니다. 오케스트레이터가 토론 흐름을 관리하며, 다음에 어떤 에이전트가 발언할지와 대화를 종료할 시점을 결정합니다. 이 패턴은 교차 기능 계획, 창의적 브레인스토밍 또는 다양한 관점이 반복적으로 상호작용하며 서로의 의견을 발전시켜야 하는 작업에 적합합니다.
