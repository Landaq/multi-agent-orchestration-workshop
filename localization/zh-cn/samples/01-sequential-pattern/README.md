# 01 顺序模式

在顺序模式中，智能体按照预定义的流水线依次工作，每个智能体的输出作为下一个智能体的输入。这种方法非常适合遵循自然递进关系的任务，例如内容创作工作流、分阶段数据转换或逐步分析。
