# 02 并发模式

在并发模式中，多个智能体同时分析相同的输入，每个智能体发挥各自的专业能力。当所有智能体完成后，它们的输出会合并为一个统一的结果。这非常适合需要多个视角同时工作的任务，例如多角度分析、集成评估或协作决策。
