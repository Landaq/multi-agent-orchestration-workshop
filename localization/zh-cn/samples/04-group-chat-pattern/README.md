# 04 群聊模式

在群聊模式中，多个智能体参与共享对话，轮流贡献各自的专业知识。编排器管理讨论流程，决定下一个发言的智能体以及对话何时结束。这非常适合跨职能规划、创意头脑风暴，或任何需要多元视角相互交流并迭代构建的任务。
